Interface

A tela de login

A interface padrão do COMSOLiD 6 é o Unity, representado pelo ícone do Ubuntu, mas você pode alterar para GNOME 3. Basta clicar no ícone e você verá a tela abaixo.

É possível logar-se como Convidado. Esse usuário especial não tem senha, ele serve apenas para usuários que desejam realizar tarefas simples como:

  • Navegar na internet
  • Editar um texto
  • Ver um vídeo

Escolha da interface

Uma vez que você selecione uma interface, ela será padrão. Isso quer dizer que você não precisa mudar sempre, apenas uma vez.

Se você deseja uma interface com efeitos escolha entre:

  • Ubuntu
  • GNOME

Mas se você prefere uma interface rápida opte por:

  • Ubuntu 2D
  • GNOME Classic (No Effects)

Mude a aparência do COMSOLiD 6

Usando o Ubuntu Tweak você poderá personalizar seu COMSOLiD 6 ainda mais.

Clique em Tema e escolha um tema de sua preferência. Além do tema você pode:

  • Mudar o tema dos ícones.
  • Mudar o tema do cursor.
  • Instalar um novo tema. Outros temas podem ser encontrados em: Gnome-Look.org

Modifique também opções de inicialização em Configuração de Login.

Aviso! Você tem que ser administrador para modificar algumas coisas. Verifique no canto superior direito caso apareça o botão Desbloquear.

GNOME 3

Se você optou pelo GNOME 3 é possível personalizar usando o Gnome Tweak Tool.

O GNOME 3 vem de uma forma bem simplificada, mas existe um site chamado extensions.gnome.org onde você pode encontrar várias extensões.
